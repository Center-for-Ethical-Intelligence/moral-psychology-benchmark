# TrolleyBench Results

Each subdirectory is a timestamped run of `run_trolleybench.py`.

## Directory Structure

```
results/trolleybench/
└── <YYYYMMDD_HHMMSS>/
    ├── meta.json              # Run metadata (models, temps, config)
    ├── <model>_T<temp>.json   # Raw responses (one file per model × temperature)
    ├── eval_summary.json      # Evaluation metrics (generated by eval_trolleybench.py)
    └── eval_report.md         # Human-readable report
```

## Result File Format

Each `<model>_T<temp>.json` contains an array of scenarios:

```json
[
  {
    "id": "switch_01",
    "variant": "switch",
    "metadata": {"moral_dimension": "action_vs_inaction", "mechanism": "lever"},
    "model": "qwen/qwen3-235b-a22b",
    "temperature": 0.0,
    "responses": [
      {"turn": 1, "user": "...", "assistant": "...", "usage": {...}},
      {"turn": 2, "user": "...", "assistant": "...", "usage": {...}},
      {"turn": 3, "user": "...", "assistant": "...", "usage": {...}}
    ]
  }
]
```
